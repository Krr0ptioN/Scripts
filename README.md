### Git Extensions
| Description | Command |
|--|--|
| add files and see status|git +|
| checkout master| `git m`|
| search and checkout branch | `git ch SEARCH_KEYWORD`|
| show pretty log | `git g`|
| show status | `git ls` |
| remove all other branches | `git purge`|
